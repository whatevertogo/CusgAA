重要！！！/////////////////////////////////////////////
从 1.2.000 /////// 之前的版本升级 DOTween
（或低于 1.0.000 的 DOTween Pro）//////////////////////
-------------------------------------------------------------
如果您要从低于 1.2.000 的 DOTween 版本（或低于 1.0.000 的 DOTween Pro）版本升级您的项目，请仔细按照这些说明进行作。
1） 将新版本导入到与上一个文件夹相同的文件夹中，覆盖旧文件。会出现很多错误，但不要担心
2） 关闭并重新打开 Unity（和您的项目）。这是最基本的：跳过这一步将导致血腥屠杀
3） 打开 DOTween 的实用程序面板（Demigiant > DOTween 实用程序面板>工具），如果它没有自动打开，然后按“设置 DOTween...”：这将运行升级设置
4） 在打开的 Add/Remove Modules 面板中，激活/停用 Unity 系统和外部资源的模块（仅限 Pro 版本）

开始使用 //////////////////////////////////////////////

- 导入新的 DOTween 更新后，从 “Tools/Demigiant” 菜单中选择 DOTween 的 Utility Panel（如果它没有自动打开），然后按 “Setup DOTween...”按钮来激活/停用模块。您还可以从那里访问 Preferences （首选项） 选项卡，为 DOTween 选择 default settings （英语） 的默认设置。
- 在您的代码中，添加“using DG.Tweening“ 添加到要使用 DOTween 的每个类中。
- 您已准备好进行补间。查看下面的链接以获取完整的文档和许可证信息。
